loqhness by solenskiner

Poker equity calculator and more

Download

You can download this project in either zip or tar formats.
You can also clone the project with Git by running:

git clone git://github.com/solenskiner/loqhness

Install

Compile the project by running:

qmake
./configure
make

Dependencies

libpoker-eval
qt
